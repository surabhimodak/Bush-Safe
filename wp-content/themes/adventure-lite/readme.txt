/* How To Set Theme Custom Features And Options */
/*-----------------------------------------------------------------------------------*/

1 - How to set slider ?

	First Under admin pannel going to Appearance >> Customize >> Static Front Page and here set option "Front page displays" as "A Static Page" And Set Page For Front Page.
	
	So for set Slider Going To Appearance >> Customize >> Slider Settings 
	
	when you reach here than slider is hide so first uncheck "Uncheck To Show Slider" Checkbox There is have 3 dropdown for three slides "Select page for slide one, Select page for slide two, Select page for slide three".
	
	So choose page for each slide option and click on "Saved" Buttons.
	
	if you don't create page for slide so going to "Pages" And Add New Page For Slide. for example you add new page with Page Title "My First Slide" there page title is slide title and Description content is slide description and featured image as Slide Image.
	
2 - How to set 6 boxes below slider ?	
	 
	First create 6 pages for 6 boxes eg: Cycling, Family, Hiking, Sailing, Skating and Wildlife .... than going to Appearance >> Customize >> "Home Section 1 Six Info Boxes" and first uncheck "Uncheck To Show This Section" checkbox and select page though you created.
	
	Here Box Page Title is Box Title and Featured Image is Box Image.  
	
3 - How to set section two three boxes ?	

	First create 3 pages for 3 boxes eg: Diverse Destinations, Value For Money, Beautiful Places than going to Appearance >> Customize >> "Home Section 2 Three Boxes" and first uncheck "Uncheck To Show This Section" checkbox and select page though you created.
	
	under field "Section Title" add your section heading here.
	
	
4 - How to set footer option ?

	For footer there have 3 options in customizer "Socical Settings", "Footer Area" and "Contact Details". 
	
	Social Settings : there you add you social link though is displaying in footer first column bottom section.  
	
	Footer Area : there is have option for "Add title for latest news feed" [This is the footer center column blog post heading]
				  "Add title for about us" [This is the footer third column heading]	
				  "About Description" [This is the footer third column description]	
				  "Add title for footer contact info" [This is the title for footer first column]
				  
	Contact Details : This all information showing in footer first column.
